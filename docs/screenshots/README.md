README.

Expected files:

| File                   | Shows                                                  |
| ---------------------- | ----------------------------------------------------- |
| `main-window.png`      | The main window with a conversation and images        |
| `model-and-template.png` | Choosing a model and a template before generating   |

1200–1600px so they stay crisp without bloating the repo.
